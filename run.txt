


запустить список скриптов по очереди


в файле settings.yaml 
SYSTEM_RUNNER: on
если on то запускаем список скриптов по очереди заново 
сразу после завершения выполнения всего списка 
MAIN_SCRIPTS = [
    {"print": "get CANDLE_TIME..."},
    "TOOLS/get_CANDLE_TIME.py",
    {"print": "get ROUND_COUNTER..."},
    "TOOLS/SYSTEM_LARGE_DELAY_IN_SECONDS.py",    
]

если значение изминилось и уже не on то останавливаем работу 
сперва доработав начатый список до конца 
после чего выполнить список финализации 1 раз тоже по очереди
SCRIPTS_FINALIZATION = [
    {"print": "get CANDLE_TIME..."},
    "TOOLS/get_CANDLE_TIME.py",
    "TOOLS/check_if_CANDLE_TIME_is_3.py",
    {"print": "get ROUND_COUNTER..."},
    "CORE/CA_check_if_CANDLE_TIME_IS_3_true.py",
    "CORE/CB_check_if_ROUND_COUNTER_is_10.py",
    "TOOLS/SYSTEM_LARGE_DELAY_IN_SECONDS.py",    
]

и только после SCRIPTS_FINALIZATION завершить работу



все настройки скрипта вверху ввиде переменных 


все пути с прямыми слэшами
